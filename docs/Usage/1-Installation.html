Installation

Installation follows the standard ExpressionEngine addon install method. Sift consists of a module and an extension.

EE2.x Installation

  1. Upload the sift folder to your system/expressionengine/third_party/ folder
  2. In your EE control panel go to Addons > Modules. Click Install on the Sift row.
  3. Installation is done. Nothing to it. Visit Usage to see what tags you need to drop into your templates.

Note: To use Sift you'll also need Matrix installed and setup on your site
