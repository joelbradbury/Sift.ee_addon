Examples

Sift can be put to use in lots of ways. The simplest of these is just a standard search form and results view, but you can also hard-code searches directly on the sift:view tag, or combine both user supplied and hard-coded search parameters.

Basic Example

Simply using the sift:form and sift:results tags to build the simplest of matrix searches.

Form Options

Setup the form inputs to list the available values for the various cell values.

Display Options

Switch up the results display.

Hardcoded Searches

Bypass the search form, and choose the sifting parameters directly. Maybe based on a member's preferences?

Merged Searches

Setup some values for the user, let them search on some others. The apple pie of searches.

Category Searching

Pass over categories to limit your searches too.
