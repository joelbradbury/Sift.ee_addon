FAQ

What about General searches?

Sift is designed to be used for filtering down data held in matrix fields. If you're looking for a more general search facility, try SuperSearch or Low Search.

What about third party fields?

If you're using the sift:form tag to populate a list of possible cell options Sift will inspect the database for possible cell values. Native fields like text and textarea work perfectly here. Also supported are fields, like P&T Checkboxes and P&T Select.

If you've got a third party fieldtype you need support for, send a request to sift@squarebit.co.uk and we'll be able to add support for you no problem!
