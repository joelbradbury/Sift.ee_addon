What can I do with Sift?

Sift gives you amazing search and filter powers on data help in matrix fields.

Briefly, the main points for Sift

  • Populate search fields with possible values for columns
  • Seperate each result row into it's own psuedo entry
  • Re-populate your search terms in search forms
  • Cache ALL the things
