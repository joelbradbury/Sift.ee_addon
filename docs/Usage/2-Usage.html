Usage

What can I do with Sift?

Sift gives you amazing search and filter powers on your matrix fields.

It's Search, for Matrix Fields

Sift searches your matrix fields for content, then returns only those rows. When you search with sift the results you get back are just the entries and row data that matches.

Free Your Product Data

Have a product site that keeps options in matrix fields? How about only showing user's the fields that are relevant to them instead? You can use Sift's template tags to hardcode direct searches and filter down rows directly.

Focused, Quick

Sift lets you narrow down your results to only those that matter. All of the normal standard EE entry tags and matrix field tags work just as you'd like.

Show Your Options

How about displaying the possible options for a matrix column in a select field? Sift's :form tag lets you populate your forms with options of valid values direct from your matrix data. No more guessing.

Sift Does More

With sift you can do lots more than just basic matrix searching. Check out the examples to get an idea of what you can do.

Don't Need Matrix Searching?

Sift is for searching and filtering your matrix field data. It doesn't search standard entry data beyond the basic native EE field searching. For entry data searches we'd recommend using SuperSearch or Low Search.
